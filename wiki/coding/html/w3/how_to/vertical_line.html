How TO - Vertical Line

Learn how to create a vertical line with CSS.

How To Create a Vertical Line

Example

 <style>
.vl {
  border-left: 6px solid green;
  height: 500px;
}
</style>

<div class="vl"></div>

How to center the vertical line in your page:

Example

.vl {
  border-left: 6px solid green;
  height: 500px;
  position: absolute;
  left: 50%;
  margin-left: -3px;
  top: 0;
}
