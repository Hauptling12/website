Screen brightness to bright (xorg)

the minimum brightness level is too bright and xbacklight is not working

command

xrandr --output eDP-1 --brightness 0.8

command break down

--brightness "0.8" - change this to increase or decrease brightness

--output "eDP-1" - change this to your monitor

to find the monitor output run command

xrandr | grep -P "($SCREEN)" | awk '{print $1}' | sed -n 2p
